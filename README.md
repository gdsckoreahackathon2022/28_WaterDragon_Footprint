# 🐾WaterDragon_Footprint🐾
탄소 발자국을 줄이는 습관을 재미있는 챌린지를 통해 자연스럽게 형성할 수 있는 앱 서비스
 
<br></br>
## 핵심 기능🙌
- **챌린지 참여** : '탄소발자국 줄이기' 챌린지 참여 & 기록 
- **챌린지 인증** : 참여 수 충족 시 타 사용자의 확인을 받기 위해 공유
- **챌린지 뱃지** : 타 사용자의 인증으로 챌린지 뱃지 획득
- **명예의 전당** : 뱃지 갯수에 따른 사용자 명예의 전당
